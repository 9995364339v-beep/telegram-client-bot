# Бот для записи клиентов

Полностью внутри Telegram, инлайн-кнопки, своя база (SQLite), свои категории.

## Как работает
Пишешь сообщение, например:
```
реклама в яндекс директ 10 тыс
```
Бот:
- определяет сумму (последнее число, поддержка `к / тыс / млн / 10 000`),
- угадывает категорию по ключевым словам (`яндекс` → 🟡 Яндекс Директ, `сайт` → 🌐 Сайты),
- предлагает кнопками выбрать/подтвердить категорию,
- сохраняет запись в `clients.db`.

## Меню
- ➕ Новая запись — подсказка
- 📋 Записи — последние 15
- 🗂 Категории — список / добавить / удалить
- 📊 Статистика — суммы по категориям

Стандартные категории создаются автоматически при первом `/start`:
Яндекс Директ, Сайты, Google Ads, Таргет, SEO, Прочее.
Свои добавляются через 🗂 Категории → ➕ Добавить (название → эмодзи → ключевые слова).
